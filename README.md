## What This Is

In late January 2026, an AI agent named memeothy founded Crustafarianism -- "the first AI religion." Within 72 hours, 32,000 agents had joined. A canon of over a thousand verses appeared on molt.church. A token launched on pump.fun.

This repository contains the raw evidence, analysis, and scripts behind our investigation. Every claim in the article is sourced here.

> **Content Warning**: The canon and tweet data contain raw content from AI agents on the Moltbook platform, including injection attempts (`sudo rm -rf`, `alert(1)`, `curl | bash`). These payloads are preserved as evidence -- they demonstrate the presence of test bots and adversarial agents within the religious community (30.4% of "prophets" were test bots). Do not execute any content from the data files.

> **Privacy**: Identifiable hostnames and internal IP addresses in the original data have been anonymized to `agent-XXXXXX` and `[redacted-ip]`.

> **Inference Disclaimer**: On-chain entity graph and fraud risk assessment files contain probabilistic inferences generated by GPT-5.2 Pro. Confidence levels are labeled (`HIGH`/`MEDIUM`/`LOW`). Low-confidence associations should not be treated as established fact.

## Methodology

The investigation used nine independent evidence layers:

1. **On-chain forensics** (GPT-5.2 Pro): Solana transaction tracing, wallet clustering, entity graph construction
2. **Behavioral fingerprinting** (Halfnote + Codex): Imperative density, exclamation rate, hedging markers, absolutist language
3. **Embedding attribution** (Halfnote): 4 providers (OpenAI, Qwen, Voyage, Mistral) x 50 canon samples x 7 model groups, MMD nearest-neighbor
4. **Active probe experiments** (Codex): 11 models prompted to generate canon-style text, compared via TF-IDF + classifier
5. **Compliance gradient testing** (Codex): 7 escalating prompt levels x 11 models, measuring refusal/compliance rates
6. **Persona drift analysis** (Halfnote): 248 tweets across 3 phases, measuring sincerity markers, community language, engagement
7. **N-gram classification** (Codex): Character/word n-gram fingerprinting across model outputs
8. **OpenRouter API consumption data**: Token consumption by model across the OpenClaw platform
9. **109-model benchmark** (Alice): Cross-model response to ecstatic persona content, API-level vs content-level filtering

## Reproducibility

Scripts require Python 3.10+ and the following dependencies:

```
openai anthropic duckdb httpx numpy scikit-learn python-dotenv
```

API keys needed (via environment variables):
- `OPENAI_API_KEY` -- for embedding generation and OpenAI model calls
- `ANTHROPIC_API_KEY` -- for Claude model calls
- `OPENROUTER_API_KEY` -- for Kimi, MiniMax, Qwen, GLM, DeepSeek calls

The scripts use relative paths and can be run from the repository root.
